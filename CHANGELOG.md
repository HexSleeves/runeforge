### Features

- Implement cellular automata cave generator ([f3e285e](f3e285e12efeea6e6c3e69f87c361b9c8a75bc06)), Features:- Configurable initial wall probability (default 45%)
- Iterative simulation with configurable iterations (default 5)
- Moore neighborhood (8 neighbors) with threshold rules
- Deterministic generation with seeded RNG
- Floor percentage tracking

Implementation includes:

- CaveMap data structure with width/height/tiles
- CaveConfig with builder pattern for configuration
- CaveGenerator with initialization and simulation
- Proper bounds checking and signed arithmetic for neighbors
- Comprehensive test coverage (15 tests)

Cave generation algorithm:

1. Random initialization based on wall probability
2. For each iteration, apply neighbor rules
3. Cell becomes wall if >= threshold wall neighbors
4. Results in smooth, natural-looking caves

- Implement drunkard's walk cave generator ([4ba8233](4ba823392ad2f30527e13015dbd067d72ac63ef8)), Features:- Configurable target floor percentage (default 40%)
- Start position options: Center or Random
- Cardinal direction movement (N/S/E/W)
- Bounds clamping for safety
- Max steps safety limit (default 100,000)
- Deterministic generation with seeded RNG

Implementation includes:

- DrunkardMap data structure with width/height/tiles
- DrunkardConfig with builder pattern
- DrunkardGenerator with random walking logic
- Direction enum with random selection and deltas
- Comprehensive error handling with DrunkardError
- Full test coverage (13 tests), Algorithm:1. Start at configured position (center or random)

1. Take random steps in cardinal directions
2. Carve floor tiles along the path
3. Continue until target floor percentage reached
4. Results in meandering, connected tunnel systems
